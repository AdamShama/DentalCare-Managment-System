# 🦷 DentalCare Management System

A full-featured **Java Swing desktop application** for managing a dental clinic, including patients, staff, appointments, treatments, inventory, and reports.

Designed with a clean UI, real database integration, and role-based access control.

---

## ✨ Features

- 🔐 Role-based login system:
  - Dentist
  - Hygienist
  - Secretary
  - Patient

- 🧑‍⚕️ Patient Management
  - Personal details
  - Medical & dental history
  - Insurance tracking

- 📅 Appointment System
  - Scheduling & tracking
  - Status updates (Paid, Refunded, etc.)
  - Reminder system

- 🦷 Treatment Plans
  - Multi-phase treatments
  - Staff assignment
  - Progress tracking

- 📦 Inventory Management
  - Stock tracking
  - Expiration dates
  - Supplier management

- 📊 Reports
  - Treatment reports
  - Invoice tracking
  - XML import/export

---

## 🛠️ Technologies Used

- Java (Swing GUI)
- JDBC (UCanAccess)
- Microsoft Access (.accdb)
- Object-Oriented Programming (OOP)
